KEY FILE DESCRIPTIONS:
======================

FRONTEND FILES:
- Authentication pages (login, signup)
- Role-based dashboards (Soldier, Command, Admin)
- Fitness test management interfaces
- Nutrition & hydration tracking
- Real-time notifications display
- Report generation & viewing
- Progress charts using Chart.js
- Responsive components with Tailwind CSS

BACKEND FILES:
- RESTful API endpoints for all modules
- JWT-based authentication & authorization
- Role-based access control (RBAC)
- PostgreSQL database integration
- Redis for caching & sessions
- Socket.io for real-time notifications
- PDF report generation
- Automated alert system (cron jobs)
- BMI & fitness score calculations
- Data analytics & insights

DATABASE TABLES:
- users (authentication)
- soldiers (personal details, service no, rank)
- fitness_tests (IPFT, monthly, quarterly tests)
- nutrition_plans (meal plans, hydration)
- injuries (injury records, recovery tracking)
- notifications (alerts, reminders)
- medical_categories (A, B, C, Temporary)
- performance_metrics (long-term tracking)

CONFIGURATION FILES:
- Environment variables (.env, .env.local)
- TypeScript configs (tsconfig.json)
- Next.js config (next.config.js)
- Docker setup (docker-compose.yml)
- Git ignore (.gitignore)
