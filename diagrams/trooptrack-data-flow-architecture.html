TroopTrack — Data Flow Diagram (DFD)

Source: diagrams/trooptrack-data-flow-architecture.mmd

How to read: Actors (Soldier / Clerk / Adjutant) interact with the Next.js UI, which calls 5 simplified backend processes (Auth, Soldier/Profile, Plans, Tests/IPFT, Reports/Analytics). Each process reads/writes the same central SQLiteCloud database tables (User, LoginSession, FitnessPlan, AssignedPlan, FitnessTest, Report, IpftDate).
